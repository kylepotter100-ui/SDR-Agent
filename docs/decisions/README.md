# Decision records

Short notes on non-obvious technical decisions. Write one whenever you make a call future-you might question.

## Format

- File name: `NNNN-short-topic.md`, four-digit zero-padded sequence.
- Two to four short paragraphs is fine. Audit trail matters more than prose.
- Date the decision in the body. Don't rewrite past records — supersede with a new one and link back.
