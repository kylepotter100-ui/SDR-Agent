# KP SDR Agent

Autonomous prospecting agent for KP Solutions. Discovers newly incorporated UK businesses in target SIC codes and postcode regions, enriches them, and ships Kyle a Monday digest with personalised first-touch email drafts.

The full project spec — mission, working method, stack, conventions — lives in [`CLAUDE.md`](./CLAUDE.md). Phase-by-phase implementation briefs live in [`briefs/`](./briefs). Decision records live in [`docs/decisions/`](./docs/decisions). Start there.

```bash
pnpm install
pnpm dev
```
